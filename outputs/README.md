# 📂 Outputs do Projeto

> Tudo que sai do projeto vai aqui. Organize por fase:

- **`discovery/`** — pesquisa, benchmark, contexto de mercado
- **`drafts/`** — versões em construção (rascunhos, wireframes)
- **`qa/`** — revisões, testes, pareceres jurídicos/compliance
- **`final/`** — entregas finais prontas pro cliente/público

> Use versões (`v1`, `v2`, …) ao invés de sobrescrever.
